Identity Broker

November 29, 2024

An identity broker is an intermediary service that facilitates the authentication and authorization of users across different identity providers.

Description

In the context of Identity and Access Management (IAM), an identity broker acts as a trusted intermediary between a user and multiple identity providers. Its main function is to streamline the authentication process by allowing users to log in once and gain access to multiple systems without needing to manage separate credentials for each one. An identity broker uses standardized protocols such as SAML, OAuth, or OpenID Connect to authenticate users and to relay identity information between different systems. This process not only enhances security by reducing password sprawl but also improves the user experience by simplifying access to various services. Identity brokers are commonly used in Single Sign-On (SSO) solutions, enabling users to access multiple applications with a single set of credentials.

Examples

  • A user logs in to their company's identity broker, which then provides access to cloud services like Microsoft 365 and Salesforce without additional logins.
  • An organization uses an identity broker to connect its internal directory service with external SaaS applications, allowing employees seamless access.

Additional Information

  • Identity brokers can help organizations enforce consistent access policies across different platforms.
  • They can also provide detailed logging and auditing capabilities, enhancing security and compliance.
