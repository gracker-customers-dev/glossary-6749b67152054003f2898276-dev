Active Directory

November 29, 2024

A directory service developed by Microsoft for Windows domain networks.

Description

Active Directory (AD) is a directory service developed by Microsoft for Windows domain networks. It is included in most Windows Server operating systems as a set of processes and services. An AD domain controller authenticates and authorizes all users and computers in a Windows domain network, assigning and enforcing security policies for all computers and installing or updating software. For example, when a user logs into a computer that is part of a Windows domain, Active Directory checks the submitted password and determines whether the user is a system administrator or a regular user. Active Directory makes use of various standardized protocols including LDAP (Lightweight Directory Access Protocol), Kerberos, and DNS (Domain Name System). It uses a hierarchical structure of objects, which can include users, computers, applications, and groups, all managed within a domain. This structure allows for a centralized management of network resources, making it easier to enforce policies, deploy software, and secure data. AD is a fundamental component of identity and access management (IAM) in many enterprise environments.

Examples

  • Example 1: A large corporation uses Active Directory to manage employee access to internal resources such as email and file servers.
  • Example 2: An IT department uses Active Directory to enforce security policies across all computers in a network, such as password complexity requirements.

Additional Information

  • Additional info 1: Active Directory can be extended with additional services such as Active Directory Federation Services (AD FS) for single sign-on and Active Directory Certificate Services (AD CS) for managing digital certificates.
  • Additional info 2: Azure Active Directory is a cloud-based implementation of Active Directory, providing additional features and integration with cloud services.
